

# LINE 圖文選單圖片生成器

## 目標
建立一個管理員工具頁面，在瀏覽器中繪製 LINE Rich Menu 圖片（2500x1686 像素），並提供下載按鈕。

## 設計
使用 HTML Canvas 繪製 2x2 四格選單圖片：
- **左上**：📝 申請休假（藍色背景）
- **右上**：📊 查詢假期（綠色背景）
- **左下**：📆 當月休假（紫色背景）
- **右下**：🌐 網頁版請假（橘色背景）

每格包含大圖示 + 中文標題，風格簡潔專業。

### 實作細節

`RichMenuGenerator.tsx`：
- 使用 `useEffect` + `canvas.getContext('2d')` 繪製 2500x1686 圖片
- 四格各有不同背景色，中央繪製白色圖示文字和功能名稱
- 提供「下載圖片」按鈕，輸出為 PNG
- 頁面僅供管理員使用，不加入主導覽列
